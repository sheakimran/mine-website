Infrastructure as Code with Terraform

Created production-grade VPCs, EC2 instances, and load balancers using Terraform on AWS with version control.

Infrastructure Automation on AWS

Designed and implemented cloud infrastructure using Terraform modules for AWS. The infrastructure included VPCs, public/private subnets, internet gateways, NATs, security groups, EC2 instances, and ALBs.

Applied GitOps methodology by storing Terraform code in GitHub and managing infrastructure via pull requests. Terraform remote state was configured in S3 with DynamoDB locking to avoid state conflicts.

Modularized the setup for reusability and scalability across multiple environments (dev, staging, production).
