Observability in Kubernetes

Built and deployed a full monitoring stack using Prometheus for metrics scraping, Alertmanager for alerting, and Grafana for visualization.

Exporters were used to gather node, pod, and application metrics while Grafana dashboards provided insights into system health and usage trends.

Configured alert rules to detect anomalies and send notifications through email and Slack integrations, improving incident response times.
